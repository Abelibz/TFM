# Datos del proyecto

Los datos utilizados en este Trabajo Final de Máster proceden de fuentes con
restricciones de uso y confidencialidad (portales inmobiliarios y datos con licencia).

Por este motivo, los conjuntos de datos no se publican en este repositorio.

No obstante, se incluye:
- El código completo de procesamiento y análisis en notebooks Jupyter.
- La documentación metodológica en la memoria y las diapositivas.
- Una aplicación web que permite explorar los resultados agregados sin exponer los datos originales.
